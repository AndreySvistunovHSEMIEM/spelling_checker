# Визуализация статистики ошибок

## Обзор

Модуль визуализации предоставляет графики и диаграммы для отображения динамики ошибок по датам. Он интегрирован в существующую систему статистики и позволяет анализировать прогресс пользователя в обучении.

## Функциональность

### Типы графиков

1. **Общее количество ошибок по дням**
   - Показывает количество ошибок, совершенных в каждый день
   - Полезно для отслеживания общей тенденции

2. **Ошибки по категориям**
   - Отображает ошибки, сгруппированные по категориям слов
   - Позволяет выявить проблемные области

3. **Прогресс (улучшение/ухудшение)**
   - Показывает накопленное количество ошибок
   - Включает скользящее среднее для выявления тенденций

### Интерактивные элементы

- Возможность выбора временного периода
- Вкладки для переключения между таблицей и графиками
- Выбор типа графика
- Интерактивные элементы на графиках (при наведении)

## Архитектура

### Классы

- `StatisticsVisualizer` - основной класс для создания графиков
- `VisualizationWidget` - Qt-виджет для отображения графиков
- `ErrorDynamicsDialog` - диалоговое окно для отображения визуализации

### Интеграция

- Интегрирован с `TrainingState.mistake_history`
- Использует существующую систему хранения данных об ошибках
- Доступен через кнопку в главном окне

## Использование

### В главном окне

1. Нажмите кнопку "Графики ошибок" (иконка с ошибкой)
2. Выберите временной период в верхней части окна
3. Переключайтесь между вкладками "Таблица ошибок" и "Графики"
4. Выберите тип графика из выпадающего списка

### В коде

```python
from ui.dialogs.statistics.visualization import VisualizationWidget

# Создание виджета визуализации
widget = VisualizationWidget()

# Отображение графика
widget.plot_errors_dynamics(
    mistake_history=training_state.mistake_history,
    start_date=start_date,
    end_date=end_date,
    chart_type="total"  # "total", "by_category", "progress"
)
```
