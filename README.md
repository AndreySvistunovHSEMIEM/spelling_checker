# Spelling Trainer

Desktop-прототип орфографического тренажера для детей с механикой мотивации через реальные награды.

## Назначение

Приложение предназначено для демонстрации гипотезы:

ребенок выполняет орфографические задания, получает баллы за правильные ответы, видит прогресс к выбранной цели и после достижения цели получает подтвержденную родителем награду.

## Стек

- Python
- PySide6

## Структура проекта

- `main.py` - точка входа
- `core/` - модели данных, константы, репозиторий, работа с медиа
- `ui/` - главное окно, меню, диалоги, иконки
- `utils/` - вспомогательные функции

## Основной сценарий

1. Пользователь открывает приложение.
2. В блоке `Моя цель` задается награда и количество баллов.
3. Ребенок запускает тренировку и прослушивает слово.
4. Ребенок вводит ответ.
5. За правильный ответ начисляются баллы.
6. Прогресс к цели обновляется на экране.
7. После достижения цели родитель подтверждает выдачу награды.

## Зависимости Linux

Если при запуске возникает ошибка:

`ImportError: libpulse.so.0: cannot open shared object file`

нужно установить системную библиотеку:

```bash
sudo apt update
sudo apt install libpulse0
```

При необходимости можно также установить:

```bash
sudo apt install libpulse0 libasound2t64 libglib2.0-0 libxkbcommon0
```

## Данные приложения

Во время локального запуска приложение использует файлы данных в `core/`:

- `core/words.json`
- `core/progress.json`
- `core/settings.json`

Также приложение использует локальные ресурсы:

- `core/images/`
- `core/audio/`
- `ui/icons/`

## Для чего были внесены изменения

В рамках учебного задания приложение было адаптировано под гипотезу о системе реальных наград:

- добавлен блок `Моя цель`;
- добавлен прогресс к цели;
- добавлено подтверждение награды родителем;
- зафиксирован режим награды в баллах для демонстрационного сценария.
